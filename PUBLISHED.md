## 📚 How to Use These Skills

### Installation
```bash
# Clone the repository
git clone https://github.com/kushalchalla981-tech/opencode-skills.git
cd opencode-skills

# Copy skills to your OpenCode installation
cp -r .opencode/skills/* ~/.opencode/skills/
```

### Usage
Each skill can be invoked with:
```bash
/<skill-name> [arguments]
```

Examples:
```bash
/loop 5m check deploy status
/batch migrate all lodash usage
/skillify capture testing workflow
```
