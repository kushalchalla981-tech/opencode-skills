## 🚀 How to Use Your Live Buddy

### Show Your Buddy
```bash
cd .opencode/skills/buddy
node buddy-generator.js show
```

### Generate a New Buddy
```bash
cd .opencode/skills/buddy
node buddy-generator.js generate
```

### Animate Your Buddy
```bash
cd .opencode/skills/buddy
node buddy-generator.js animate
```

### View Buddy Stats
```bash
cd .opencode/skills/buddy
node buddy-generator.js stats
```
